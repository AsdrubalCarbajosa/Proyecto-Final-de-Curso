# Proyecto-Final-de-Curso

## Identificación
- Título del proyecto: Investigación sobre el uso del software en el ámbito profesional y personal
- Nombre: Asdrúbal Manuel Carbajosa Sández
- Curso: 2º de Administración de Sistemas Informáticos en Red (2ºASIR)
## Objetivos
Profundizar en posibles situaciones de uso de software en el ámbito personal y profesional y estudiar qué tipo de software es más conveniente para el usuario u organización según la situación económica, la siituación geográfica y 
las distintas funcionalidades de cada software para que, en caso del ámbito empresarial, el TCO (Coste total de propiedad) sea el menor posible.
